# TaskFlow

TaskFlow es una aplicación sencilla de lista de tareas (todo list) construida con HTML, CSS y JavaScript. Permite crear tareas, marcarlas como completadas, buscarlas, filtrarlas y ordenarlas, manteniendo el estado (tareas + tema) en el almacenamiento local del navegador.

---

## ✨ ¿Qué hace TaskFlow?

- Permite añadir tareas rápidas desde un formulario.
- Marca tareas como completadas o elimínalas.
- Filtra tareas por el estado (todas / pendientes / completadas).
- Busca tareas en tiempo real mientras escribes.
- Ordena tareas alfabéticamente o por las más recientes.
- Mantiene el tema (claro/oscuro) y las tareas guardadas al recargar la página.

---

## ✅ Nuevas funcionalidades añadidas

1. **Buscador en vivo**: filtra tareas mientras tecleas en el campo de búsqueda.
2. **Botones de filtrado**: muestra solo tareas "Todas", "Pendientes" o "Completadas".
3. **Modo claro/oscuro**: botón para cambiar tema y guardar la preferencia en `localStorage`.
4. **Ordenación de tareas**: menú desplegable para ordenar tareas "A-Z", "Z-A" o "Más nuevas".

---

## 🚀 Instalación y uso

1. Clona el repositorio o descarga los archivos.
2. Abre `docs/ai/index.html` en tu navegador.
3. Empieza a añadir tareas en el campo de entrada y usa las funcionalidades de filtro, búsqueda, ordenación y tema.

> No se requiere backend; las tareas se almacenan en `localStorage`.

---

## 🛠️ Tecnologías usadas

- **HTML** (estructura de la aplicación)
- **CSS** (estilos y temas claro/oscuro)
- **JavaScript** (lógica de tareas, filtros, ordenación y persistencia)
- **LocalStorage** (persistencia de tareas y tema)
